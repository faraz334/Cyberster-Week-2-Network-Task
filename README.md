# Cyberster Week 2 Network Task

## Project Overview
This repository contains my Week 2 internship task for Cyberster Network Security.  
The task includes designing a basic network in Cisco Packet Tracer and documenting Ethernet cable Standard B.

## Task Objectives
- Create a network topology using routers, switches and PCs.
- Configure LAN-based communication.
- Use IP addressing and subnetting.
- Document Ethernet cable Standard B.
- Prepare a professional internship task report.
